# EconCausal

EconCausal is a context-aware causal reasoning benchmark for evaluating
large language models in socio-economic domains.

## Dataset
The dataset consists of causal triplets (treatment, outcome, sign)
annotated with rich socio-economic context, extracted from
high-quality peer-reviewed economics and finance research.

## Tasks
- Task 1: Causal Sign Identification
- Task 2: Context-Dependent Sign Prediction
- Task 3: Misinformation-Robust Sign Prediction

## Data Usage
This repository releases structured triplet annotations and task
splits. Original paper texts and verbatim evidence paragraphs are not
redistributed due to copyright restrictions.
